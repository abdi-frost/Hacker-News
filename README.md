# Hacker News Clone

![Hacker News Clone](https://via.placeholder.com/800x400.png?text=Hacker+News+Clone) <!-- Replace with your own image URL -->

This project is a **Hacker News** clone built with **Angular** and powered by **NGXS** for state management. It leverages free public APIs to fetch the latest stories, comments, and user details, mimicking the core functionality of the popular Hacker News platform.

## Key Features
- 🚀 **Real-time data**: Fetches latest stories and updates via free public APIs.
- 📱 **Responsive design**: Fully responsive interface for both mobile and desktop.
- 🛠️ **State management**: Efficient data handling with NGXS.
- 🔄 **Interactivity**: Browse, read, and interact with stories and comments.

## Technologies Used
- **Frontend**: Angular
- **State Management**: NGXS
- **API**: Free public APIs (Hacker News API)
- **Styling**: Custom CSS & Angular Material (or Tailwind, if used)
